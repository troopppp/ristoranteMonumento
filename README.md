# Ristorante Monumento dal 1884 - concept Angular site

Concept site in Angular per il ristorante "Ristorante Monumento dal 1884" con pagine dedicate
(router): Il Ristorante, Storia, Menù, Photo Gallery, Contatti, Eventi, Prenota. Header con logo e
CTA, palette chiara giocata su grigi e bianchi e immagini integrate nei layout.

## Setup

1. Assicurati di avere Node 18+.
2. Installa le dipendenze: `npm install`
3. Avvia in sviluppo: `npm start` (equivale a `ng serve`)
4. Build di produzione: `npm run build`

Il design è stato pensato per mettere in evidenza le informazioni di Maps: rating, fascia
prezzo, servizi, accessibilità, orari e contatti.
